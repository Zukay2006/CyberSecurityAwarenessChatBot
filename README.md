# CyberSecurity Awareness Bot

- PROJECT DESCRIPTION:
  
- This project is a C# console- based chatbot that teaches usesr about cybersecurity focusing on topics such as phishing, password safety and suspicious links. 

- FEATURES:
  
  1. ASCII art chatbot animation
  2. Voice greeting using WAV file
  3. Interactive conversation system
  4. Topic-based learning (Phishing, Passwords, Links)
  5. Input validation and user-friendly responses

- HOW TO RUN THE CODE OR PROGRAM:
  
1. Open CyberSecurityBot.sln in Visual Studio
2. Build the project
3. Run the program
4. Ensure welcome.wav is in the project folder
